IntelliJ IDEA 단축키 15가지 ✋➕✋➕✋

IntelliJ IDEA는 편집, 탐색, 리팩토링, 디버깅 등 일상적으로 수행하는 대부분의 작업에 대하여 해당 명령어의 키보드 단축키를 제공한다.

Alt + Enter F2 Alt + 1 Esc ctrl + E
코드에서 오류가 발견된 경우 해당 오류 위에 커서를 놓고 해당 단축키를 누르면 문제에 대한 수정 제안 목록이 표시된다. 에디터에서 오류와 경고 사이를 이동할 때 마우스를 사용하고 싶지 않다면, F2를 사용하여 다음 오류, 경고 또는 제안으로 점프할 수 있다. 도구 창을 열 때 사용한다. 에디터에 다시 포커스가 위치하게 한다. 최근 실행했던 파일을 확인할 수 있다.
Ctrl + B Alt + F7 tap ctrl twice Ctrl+W or Ctrl + Shift + W ctrl + /
코드 안에서 다른 코드로 이동하고 싶을 때 사용한다. 예를 들어, 필드 위에서 이 단축키를 누르면 커서가 해당 필드의 선언으로 이동한다. Alt+F7을 누르면 어떤 항목이 사용된 모든 곳이 표시된다. 어디에서든 Ctrl 키를 두 번 눌러서 어떤 항목이든 실행할 수 있다. Ctrl+W 또는 Ctrl+Shift+W(Windows/Linux)를 사용해 커서 근처의 코드 선택 영역을 확대하거나 축소할 수 있다. 주석처리할 때 사용한다.
Shift + Ctrl + Enter Ctl + Alt + L Shift+Ctrl+Alt+T Shift + Ctrl + A tap shift twice
현재 구문을 완성할 수 있다. 해당 프로젝트의 표준에 맞도록 현재 파일의 서식을 지정할 수 있다. 심볼 또는 선택 영역 위에서 이 단축키를 누르면 사용 가능한 리팩토링 옵션이 표시된다. 모든 액션을 검색할 수 있다. 모든 항목을 검색할 수 있는 검색 상자가 열린다.
